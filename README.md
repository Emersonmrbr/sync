# Núcleo MAP

## Sobre o Projeto

O site [Núcleo MAP](http://nucleomap.com.br) é dedicado à automação industrial, oferecendo uma vasta gama de soluções, incluindo máquinas CNC, controle numérico computadorizado e automação residencial. A empresa tem como objetivo integrar inteligência, eficiência e inovação para transformar a manufatura e os negócios.

## Funcionalidades do Site

- **Máquinas CNC:** Equipamentos de usinagem de alta precisão.
- **Controle Numérico Computadorizado:** Sistemas automatizados para processos industriais.
- **Automação Residencial:** Soluções tecnológicas para o controle e automação de residências.
- **Consultoria Técnica:** Suporte especializado para implementação de soluções industriais.
- **Venda e Instalação:** Serviços completos de venda e instalação de equipamentos.
- **Suporte Técnico:** Assistência contínua para manutenção e otimização de sistemas.

## Tecnologias Utilizadas

- **HTML5** e **CSS3** para a estrutura e estilo das páginas.
- **JavaScript** para interatividade e funcionalidades dinâmicas.
- **PHP** para o desenvolvimento do lado do servidor e manipulação de dados.
- **Frameworks e Bibliotecas** modernas para melhorar a performance e a experiência do usuário.

## Estrutura do Projeto

A estrutura do projeto é organizada da seguinte forma:

- **index.html:** Página principal do site.
- **sobre.html:** Página sobre a empresa e sua missão.
- **servicos.html:** Detalhes dos serviços oferecidos.
- **contato.html:** Formulário de contato e informações da empresa.
- **assets/**: Diretório contendo arquivos de mídia como imagens e vídeos.
- **css/**: Diretório contendo os arquivos de estilos CSS.
- **js/**: Diretório contendo os scripts JavaScript.
- **php/**: Diretório contendo os scripts PHP.
